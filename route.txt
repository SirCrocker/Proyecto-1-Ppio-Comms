Para ir empezando en el proyecto (idea):
1.- Crear servidor e interfaz de usuario con socket.
2.- Importar base de datos de usuario (archivo .csv).
3.- Implementar "pandas" (librería de Python) para conectar directamente a la 
base de datos y hacer las consultas cuando sea necesario (definir funciones en
el programa).
4.- Desarrollar la "lectura" de mensajes del usuario para entregar la respuesta
(usando una librería como "Tensorflow").
5.- Generar la comunicación entre el servidor y el usuario de forma efectiva (socket).
6.- Crear la función para mandar a chatear al usuario con un ejecutivo
